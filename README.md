# Инструкция по формированию документов в формате TEI (XML)

Данное руководство описывает процесс преобразования текстовых отчетов в размеченный формат TEI с использованием Python-скриптов и нейросетей.

## 1. Подготовка среды в Google Colab

Для удобства и во избежание установки локальных библиотек рекомендуется использовать сервис **Google Colab**.

1. Зайдите в Google Colab и создайте новый ноутбук.
2. Вставьте в отдельные блоки кода следующие скрипты:
    * **Блок 1:** `1 build_sample_tei_latest.py`
    * **Блок 2:** `2 extract_to_types.py`
    * **Блок 3:** `2.1 inject.py`
    * **Блок 4:** `3 extract_dates.py`
    * **Блок 5:** `3.1 injectdates.py`

**ВАЖНО ПОМНИТЬ:** Почти в каждом скрипте указаны абсолютные пути к файлам. Вам необходимо вручную изменить их на относительные пути, соответствующие вашей файловой структуре.

## 2. Подготовка файловой структуры

Создайте в Colab директорию `data/reports_txt`. В эту папку необходимо загрузить текстовый файл (`.txt` с плейн текстом), с которым вы будете работать.

![Структура папок в Colab](img/colab_files.png)

*Примечание: эталонные файлы находятся в репозитории по пути `data/reports_txt`.*

## 3. Генерация базового TEI-документа

Перед запуском первого блока необходимо внести изменения в код:
Найдите строку с параметром `--doc-index` и замените значение `default` на номер вашего документа (номер можно посмотреть в манифесте файлов):

```python
parser.add_argument("--doc-index", type=int, default=5, help="Process only one specific document index (0-based).")
```

После этого запустите **Первый блок**. На выходе в папке `tei_reports` появится документ в формате `Tei(xml)`.

![Результат генерации XML](img/tei_reports.png)

## 4. Обработка таблиц

Это наиболее трудоемкая часть процесса, требующая точности.

![Пример таблиц в PDF](img/table_example.png)

1. Найдите все таблицы в исходном PDF-документе.
2. Сделайте скриншоты каждой таблицы и сохраните их отдельно.
3. Перейдите в [Google AI Studio](https://aistudio.google.com/).
4. Загрузите скриншоты таблиц и попросите нейросеть преобразовать их в формат **CSV**.
5. Создайте в Colab директорию `tables/` и загрузите туда полученные CSV-файлы.
6. В начале вашего TEI(xml) файла добавьте ссылки на таблицы в секции `<listBibl>`:

```xml
<listBibl type="csv-tables">
    <bibl xml:id="table_csv_table1">
        <ref target="tables/table1.csv" type="csv">table1.csv</ref>
    </bibl>
</listBibl>
```

7. Пройдите по всему тексту XML-файла и в местах, где должны быть таблицы, добавьте ссылки на них:

![Пример вставки ссылок на таблицы](img/xml_table_links.png)

8. Проверьте выделенные сущности вручную.
